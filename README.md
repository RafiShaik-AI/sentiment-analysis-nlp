# sentiment-analysis-nlp
Natural Language Processing project that performs sentiment analysis on text reviews using machine learning. Includes text preprocessing, feature extraction using TF-IDF, model training with Logistic Regression, and evaluation using Python and scikit-learn

# Sentiment Analysis using Machine Learning

## Description
This project performs sentiment analysis on text reviews to classify them as positive or negative. It demonstrates basic NLP techniques such as text preprocessing and TF-IDF feature extraction.

## Technologies
- Python
- scikit-learn
- TF-IDF Vectorizer
- Google Colab

## Model
Logistic Regression

## Result
The model successfully classifies sentiment from text reviews.
